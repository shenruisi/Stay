# my-vue3-plugin

> 使用 vue3 版本进行 chrome 浏览器插件开发

## 此项目版本

```
@vue/cli 4.5.7
yarn 1.22.10
```

## node_modules 安装

```
yarn install
```

> 如果遇到问题报错，可以查看[使用 vue-cli 创建 vue3.x 版本项目报错](https://guoqiankun.blog.csdn.net/article/details/111993759) 这篇文章

## 插件开发

```
yarn run watch
```

## 插件打包

```
yarn run build
```
